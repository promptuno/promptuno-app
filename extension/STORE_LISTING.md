# Chrome Web Store Listing Draft

## Title

Promptuno - Better Prompts

## Short Description

Generate better AI prompts anywhere you work.

## Long Description

Promptuno is a clean, premium assistant for people who work with AI seriously. It helps you generate better prompts across CMD, Image, Code, and Vibe categories.

Official web app: https://www.promptuno.chat/

Open the extension from your browser toolbar or use the subtle inline Promptuno button near supported text fields. Generate a better prompt for ChatGPT, Claude, Gemini, or Copilot without leaving the page.

Promptuno stays focused on prompts first. The extension is built to reduce friction, not add a noisy toolbox to your browser.

## Key Features

- CMD, Image, Code, and Vibe prompt categories.
- Works inside major AI tools and supported text areas.
- Clean popup panel for prompt generation.
- Primary model focus: ChatGPT, Claude, Gemini, and Copilot.
- Prompt history and saved prompts foundation.
- Same 5 free generation limit as Promptuno web.
- Calm Pro paywall for more in-context generation.
- Lightweight, subtle, and designed to stay out of your way.

## Privacy-Friendly Wording

Promptuno only processes text you choose to improve or generate. Local usage count, prompt history, and saved prompts are stored in Chrome storage on your device. The extension does not require account creation in this version. AI generation requests are sent to the configured Promptuno generation provider.

## Onboarding Bullets

1. Pin Promptuno to your browser toolbar.
2. Open ChatGPT, Claude, Gemini, Copilot, Gmail, Outlook, or any page with a text field.
3. Type or select rough text.
4. Click Promptuno to improve it.
5. Insert, copy, or save the result.
